# 产品需求文档 (PRD)

## 产品概述

**产品名**：MultiNote（双人计划本）

**核心价值**：两个人各自拥有无限长卷笔记本，分享房间码后互相可见对方笔记本、可在上面涂鸦，同时支持 Notion 式块级内容组织和 GoodNotes 式手写体验。

## 用户故事

1. 我和朋友各自打开网页 → 一人创建房间 → 另一人输入房间码加入
2. 画布左侧显示朋友的当前页，右侧显示自己的当前页
3. 各自独立翻页、书写
4. 可以翻阅对方笔记本并在上面涂鸦
5. 涂鸦存储在独立图层，可随时显示/隐藏

## 功能清单

### Phase 1 — MVP

| 模块 | 功能描述 |
|------|----------|
| 房间系统 | 创建房间（随机码）、加入房间（输入码）、在线状态 |
| 双页画布 | 左=朋友当前页 / 右=我的当前页，B5 宽度，纵向无限滚动，24px 缝隙 |
| 块类型 | 段落、H1/H2/H3、待办事项、图片 |
| 手写引擎 | 块级 Canvas，钢笔（压感粗细），颜色/粗细选择 |
| 橡皮擦 | 笔画级擦除 |
| 页面管理 | 添加/翻页，独立翻页 |
| 左侧大纲 | 双笔记本目录，折叠展开，点击跳转 |
| 涂鸦层 | 存为独立图层，全局/单页切换显示 |
| 实时同步 | Yjs + WebRTC P2P 同步 |
| 本地存储 | IndexedDB 自动保存（每 5 秒） |
| 主题 | 黑白灰 + #A78BFA 浅紫，浅色/深色模式 |
| 响应式 | 平板横屏（主力）、电脑、手机基础支持 |
| 撤销重做 | 笔迹和文本 |
| 对方状态 | 页码旁圆点 + 当前页码 |

### Phase 2

- 表格块（Notion 式数据表格 + 手写绑定）
- 套索工具（框选/移动/缩放/删除/转文本）
- 更多笔刷（圆珠笔、荧光笔）
- 页面缩放（双指 50%-200%）
- 块拖拽排序
- 更多块类型（引用、分隔线、绘图块）
- 手机深度适配

### Phase 3

- 翻页动画、笔刷切换动画
- 虚拟化渲染性能优化
- 笔记本导出 (PDF/JSON)
- 块转悬浮块
- 页面排序
- PWA 支持

## 非功能性需求

- 无需注册/登录
- 纯浏览器端存储（IndexedDB）
- 无需后端服务器
- 联网使用（不要求离线）
- 配色：黑白灰 + #A78BFA 浅紫
- 深色模式跟随系统
